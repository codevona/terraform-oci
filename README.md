# Terraform OCI

## Env

### if you want to using in local 

* `cp terraform.tfvars.example-local terraform.tfvars`

### if you want to using in oci shell

* `cp terraform.tfvars.example-oci terraform.tfvars`

## Setup

### Init and dowload Packages
 * terraform init

### Review Config

* terraform plan

### Deploy

* terraform apply
